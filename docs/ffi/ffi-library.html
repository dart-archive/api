Higher level access to the FFI library.

This package is a set of functions and classes that builds on the lower level FFI library (dart:dartino.ffi).

Functions

cStringToString(ptr) → String

Converts a C string to a String in the Dartino heap. This call expects a null terminated string. The string will be decoded using a UTF8 decoder.

memoryToString(ptr, int length) → String

Converts a C memory region to a String in the Dartino heap. The string will be decoded using a UTF8 decoder.

Classes

CircularByteBuffer

A circular buffer used from both dart and c. The buffer has the start index and the end index as the first two 4 byte integers in the underlying foreign memory, and the size as the next 4. This is not thread safe, only access this from either c or dart at any point in time.

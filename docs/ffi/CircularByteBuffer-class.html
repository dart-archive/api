A circular buffer used from both dart and c. The buffer has the start index and the end index as the first two 4 byte integers in the underlying foreign memory, and the size as the next 4. This is not thread safe, only access this from either c or dart at any point in time.

Constructors

CircularByteBuffer(int size)

Creates a new buffer capable of holding size bytes. The underlying memory has 8 additional bytes for holding the head, tail and size, and one more byte to distinguish empty from full.

Dart Core

dart:collection

Classes and utilities that supplement the collection support in dart:core.

dart:core

The dart:core library for embedded dartino.

dart:dartino
dart:dartino.ffi

A low-level 'foreign function interface' library that allows Dart code to call arbitrary native platform code defined outside the VM.

dart:dartino.os
dart:dartino.service
dart:math

Mathematical constants and functions, plus a random number generator.

dart:typed_data

Lists that efficiently handle fixed sized data (for example, unsigned 8 byte integers) and SIMD numeric types.

charcode

charcode

Defines symbolic names for character code points.

charcode.ascii.dollar_lowercase

Declare integer constants for each ASCII character.

charcode.htmlentity.dollar_lowercase

Character codes based on HTML 4.01 character entity names.

dartino

dartino

Dartino utility package.

ffi

ffi

Higher level access to the FFI library.

file

file

File access APIs. Only supported when Dartino is running on a Posix platform.

gpio

gpio

GPIO support providing access to controlling GPIO pins.

gpio_mock

Mock implementation of the GPIO interface.

http

http

HTTP client implementation.

i2c

hts221

API for HTS221 "capacitive digital sensor for relative humidity and temperature" chip using the I2C bus.

i2c

I2C support for Linux.

lis3mdl

API for LIS3MDL "Digital output magnetic sensor: ultra-low-power, high-performance 3-axis magnetometer" chip using the I2C bus.

lps25h

API for LPS25H and LPS25HB "MEMS pressure sensor: 260-1260 hPa absolute digital output barometer" chip using the I2C bus.

lsm6ds0

API for LSM6DS0 "iNEMO inertial module: 3D accelerometer and 3D gyroscope" chip using the I2C bus.

lsm9ds1

API for LSM9DS1 "iNEMO inertial module: 3D accelerometer, 3D gyroscope, 3D magnetometer" chip using the I2C bus.

mbedtls

mbedtls

TLS support, based on mbedtls. This can be used in the same way as a normal Socket (and passed to the http package).

mqtt

mqtt

MQTT client library for the MQTT protocol, a lightweight IoT pub/sub messaging protocol.

nucleo_iks01a1

nucleo_iks01a1

Support library for accessing the sensors on a X-NUCLEO-IKS01A1 expansion board.

os

os

Access to the operating system. Only supported when Dartino is running on a Posix platform.

raspberry_pi

raspberry_pi

Access to Raspberry Pi 2 specific hardware features such as onboard LEDs. Also provides an API to the Sense HAT shield.

raspberry_pi.sense_hat

API for accessing the devices on the Raspberry Pi Sense HAT add-on board. See: https://www.raspberrypi.org/products/sense-hat/.

socket

socket

Dartino implementation of TCP and UDP sockets.

stm32

stm32

List divide(dynamic fn(argument), List arguments)

Divide the elements in arguments into a matching number of processes with fn as entry. The current process blocks until all processes have terminated.

The elements in arguments can be any immutable (see isImmutable) object.

The function fn must be a top-level or static function.

The ForeignLibrary main is used for looking up functions in the libraries linked in to the main Dartino binary.

Static Methods

bundleLibraryName(String libraryName) → String

Provides a platform specific location for a library relative to the location of the Dartino vm. Takes the name without lib in front and returns a platform specific path. Example, on linux, foobar_hash become PATH_TO_EXECUTABLE/lib/libfoobar_hash.so.

ForeignLibrary.fromName(String name, {bool global: false})

Looks up a foreign library by name. If the global flag is set we use RTLD_GLOBAL that will enable later lookups to use code from this library.

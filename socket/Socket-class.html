read(int bytes) → ByteBuffer

Read bytes number of bytes from the socket. Will block until all bytes are available. Returns null if the socket was closed for reading.

readNext([int max]) → ByteBuffer

Read the next chunk of bytes. Will block until some bytes are available. Returns null if the socket was closed for reading.

shutdownWrite() → void

Close the socket for writing. After the socket is closed for writing, write to the socket will fail.

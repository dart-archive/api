API to the Raspberry Pi Sense HAT add-on board.

Instantiating this class will open the I2C bus to communicate with the devices for the board.

The properties hts221, lps25h and lsm9ds1 provide access to the sensor APIs. The property ledArray provide access to the 8 x 8 LED array.

For the most common sensor APIs, this object provide direct access to some of the measurements, e.g. through the methods readTemperature and readHumidity.
